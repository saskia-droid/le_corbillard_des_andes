# Le Corbillard des Andes 

ROM d'un jeu Game Boy Advance réalisé pour l'Epic Game Jam 2026. 

Le Corbillard des Andes est prêt pour faire face à la Prophecy imminnente des chinchillas qui tombent du ciel. 

### Contrôles

- start: commencer le jeu
- croix multidirectionnelle: déplacer le Corbillard 
- A: prendre un chinchilla mort 
- B: enterrer un chinchilla mort
